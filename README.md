## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/endlessalpacaYT/Arcane-Backend-V3/
```

2. **Install packages**

```bash
npm install
```

3. **Configure environment variables:**

Rename `.example.env` to `.env` and provide the necessary information.

5. **Run**

```bash
node index.js
```
